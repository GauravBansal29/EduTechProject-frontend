# EduTechProject-frontend
A Learning Management system where users can buy affordable courses from instructors with a ad-free learning experience and instructors can create , edit , update their courses.

<h4><u>Tech Stack</u></h4>
<div><b>FrontEnd:</b>  NextJS, CSS, Bootstrap, Ant Design</div>
<div><b>Backend:</b>  Node.js, MongoDB, Express.js, REST API's, AWS SES, AWS S3, Razorpay Payment API's </div>

<h4>Major Highlights</h4>

1. Payments from users and Payouts to Instructors (Razorpay Payment API’s)
2. Email OTP based Forgot Password functionality  (Amazon Simple Email Service)
3. Scalable Video, Image Storage and Deletion (Amazon S3 cloud storage)
4. Routes and Data Protection (JWT authentication and CSRF protection)
